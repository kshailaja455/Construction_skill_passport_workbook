# Construction Worker Skill Passport

#  Project Status
 Currently in Design Phase (Workbook Completed)

#  Overview
This project proposes a digital platform to create verified skill passports for construction workers using image-based portfolios and QR verification.

#  Problem Statement
Construction workers lack formal proof of skills, leading to unfair hiring practices.

#  Proposed Solution
1. Digital worker profiles
2. Upload project images
3. QR-based supervisor verification
4. Skill tagging using image recognition

#  Features (Planned)
1. Worker portfolio system
2. Contractor search system
3. Rating & feedback mechanism
4. Low-network mobile support

#  Future Work
1. Mobile app development
2. Backend system
3. AI-based skill detection
